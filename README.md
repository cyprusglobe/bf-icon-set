# bf-icon-set
Balloon Fiesta Font Icon Set

```
put the font directory in the root of the folder
```

```
update your style.css with a relative path to the fonts

src:url('fonts/bf-icon-set.eot?stkrh7');
src:url('fonts/bf-icon-set.eot?#iefixstkrh7') format('embedded-opentype'),
		url('fonts/bf-icon-set.woff?stkrh7') format('woff'),
		url('fonts/bf-icon-set.ttf?stkrh7') format('truetype'),
		url('fonts/bf-icon-set.svg?stkrh7#bf-icon-set') format('svg');
```

```
Add the style.css file to your html file

<link rel="stylesheet" href="style.css">
```
